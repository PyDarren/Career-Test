## 一、测评体系总览

### 1.1 核心理论模型
| 模块 | 理论基础 | 题量 | 测量维度 | 产出物 |
|------|---------|------|---------|--------|
| 人格特质模块 | IPIP大五人格模型（公有领域） | 50题 | 开放性(O)、尽责性(C)、外向性(E)、宜人性(A)、神经质(N) | 五维百分位得分、32种人格原型 |
| 职业兴趣模块 | 霍兰德RIASEC理论（自主命题） | 30题 | 现实型(R)、研究型(I)、艺术型(A)、社会型(S)、企业型(E)、常规型(C) | RIASEC三字母兴趣码 |
| **合计** | - | **80题** | 11个维度 | 人格画像+职业兴趣双维度分析 |

### 1.2 计分规则
- **作答方式**：五点李克特量表（1=非常不符合，2=比较不符合，3=一般，4=比较符合，5=非常符合）
- **反向计分**：部分题目采用反向表述，计分前需反转（1↔5，2↔4）
- **维度得分**：各维度题目得分取均值，转换为百分位（基于本土常模）
- **原型判定**：五维度按百分位50%为界，高于=高倾向(↑)，低于=低倾向(↓)，组合为32种原型
- **RIASEC码**：六维度得分排序取前三，生成三字母兴趣码

### 1.3 产品分层策略
| 层级 | 内容 | 商业模式 |
|------|------|---------|
| **免费层** | 人格画像名 + RIASEC码 + 色彩光谱 + 五维雷达图 + 一句话总结 + 核心特质3条 + 社交建议 + 高清认证卡 | 引流 + 社交裂变 |
| **付费层（2.99元）** | 12章万字深度报告：画像总览、维度详解、稀有度、同类名人、优势劣势、成长建议、恋爱专题、职业路径、推荐岗位、协作指南、压力预警、行动计划 | 变现核心 |

### 3.3 维度符号说明
- **O↑** = 开放性高分（>50百分位）：好奇、创新、想象力丰富
- **O↓** = 开放性低分（≤50百分位）：务实、传统、偏好确定性
- **C↑** = 尽责性高分：自律、有条理、可靠
- **C↓** = 尽责性低分：灵活、随性、不拘小节
- **E↑** = 外向性高分：活跃、健谈、社交驱动
- **E↓** = 外向性低分：内敛、安静、独处充电
- **A↑** = 宜人性高分：友善、合作、同理心强
- **A↓** = 宜人性低分：独立、直接、竞争性强
- **N↑** = 神经质高分：敏感、情绪波动大、易焦虑
- **N↓** = 神经质低分：稳定、冷静、抗压能力强

### 第9章 职业发展深度专题

#### 9.1 RIASEC兴趣码解析
**IRC（研究型-现实型-常规型）** ——"研究+实践+秩序"三位一体
- **研究型(I)**：分析、研究、解决复杂问题，享受智力挑战
- **现实型(R)**：动手操作、使用工具、解决实际问题，不满足于纯理论
- **常规型(C)**：组织、整理、处理结构化信息，在复杂中建立秩序

**核心职业驱动力**：用研究型的好奇心发现问题，用现实型的动手能力构建方案，用常规型的秩序感将其系统化标准化。天然适合"系统架构"类角色。

#### 9.2 职场画像
- **工作风格**：安静高效的深度贡献者，不追求焦点但产出质量高
- **理想环境**：足够安静的深度思考时间、同事专业互相尊重、问题足够复杂、有明确质量标准
- **反感环境**：频繁被打断的开放式办公、政治斗争氛围、只讲关系不讲能力、低标准文化

#### 9.3 三条核心职业路径

**路径一：技术架构纵深线**
- 初级工程师 → 高级工程师 → 技术专家 → 系统架构师 / 首席工程师
- 匹配度：★★★★★
- 核心优势：高开放保证技术学习力，高尽责保证方案可靠性，低外向适配深度技术工作

**路径二：研究工程复合线**
- 研发工程师 / 应用科学家
- 匹配度：★★★★☆
- 核心优势：横跨研究与工程，既能做前沿探索又能工程化落地

- **身体信号**：失眠、头痛、肠胃不适、持续疲劳
- **行为信号**：比平时更沉默、工作效率下降、对原本感兴趣的事提不起劲
- **认知信号**：完美主义加剧、决策变慢、反复纠结细节

#### 12.2 高压场景应对策略
1. **强制断连**：设定每日"无工作时间"，物理上离开工作环境
2. **分解任务**：将大目标拆解为最小可执行单元，逐个击破
3. **寻求支持**：不需要独自扛，主动向同事或上级求助不是软弱
4. **调整预期**：非常时期接受"足够好"，不必追求完美

#### 12.3 长期心理健康维护
- 保持规律作息和运动，你的低神经质不代表身体不需要恢复
- 建立情感表达渠道：写作、倾诉、艺术创作均可
- 定期进行"意义感检查"：确认当前工作与你的价值观是否一致
- 维护深度人际关系：高质量的社会支持是心理韧性的基石

## 附录：评分计算方法说明

### A.1 大五人格维度计分
1. 反向题先反转得分：原始分为x，反转后得分 = 6 - x
2. 各维度10道题得分相加，计算均值（1-5分）
3. 对照常模转换为百分位得分（0-100%）
4. 以50百分位为界，判定高(↑)/低(↓)倾向

### A.2 RIASEC兴趣码计分
1. 每维度5道题，直接求和（5-25分）
2. 六维度按得分从高到低排序
3. 取前三位生成三字母兴趣码（如IRC、AES等）

- 输入：五维度高低倾向（共32种组合）
- 输出：对应编号的人格原型名称及全套报告内容
- 32种原型一一对应，无重叠无遗漏
