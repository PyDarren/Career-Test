## 1. 文档概览

### 1.1 编写目的

本文档为「画己职测」项目的完整测试用例矩阵，按 PRD 功能模块划分测试套件，覆盖核心链路、边界场景与异常场景，用于指导功能测试、集成测试与安全测试的执行。

### 1.2 模块缩写说明

| 缩写 | 模块名称 | 说明 |
|------|---------|------|
| HOM | 首页与引导流程 | Landing Page、测评引导页 |
| ANS | 答题模块 | 动态答题、进度可视化、断点恢复、效度检测 |
| RES | 测评结果与免费报告 | 免费结果页、三层标签、职业推荐 |
| RPT | 深度报告 | 付费解锁、12 章节内容展示 |
| PAY | 支付流程 | 微信支付、支付宝、订单状态轮询、六道防线 |
| CAR | 人格认证卡生成与分享 | 认证卡生成、裂变分享 |
| ADM | 后台管理 | 题库管理、订单管理、数据看板、内容配置 |
| TRK | 埋点上报 | 8 种行为事件追踪 |
| ALG | 算法与数据 | OCEAN 计分、RIASEC 计分、画像匹配、数据完整性 |
| SEC | 安全 | SQL 注入、XSS、CSRF、越权访问、数据加密 |

### 1.3 优先级定义

| 优先级 | 含义 | 覆盖要求 |
|--------|------|---------|
| P0 | 核心链路用例，必须 100% 通过 | 所有核心功能路径 |
| P1 | 重要功能用例，通过率 ≥ 95% | 辅助功能、重要边界 |
| P2 | 一般功能用例，通过率 ≥ 90% | 体验优化、次要边界 |
